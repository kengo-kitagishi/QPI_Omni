# to_uint8正規化の改善計画

## 問題点

現在の`to_uint8()`関数（24-32行）：

```python
def to_uint8(img):
    img_min = np.min(img)
    img_max = np.max(img)
    normalized = (img - img_min) / (img_max - img_min)
    return (normalized * 255).astype(np.uint8)
```

**問題**:

- 各画像ごとに最小値・最大値が異なる
- 同じピクセル値でも画像によって異なるuint8値にマッピング
- 外れ値の影響を受けやすい
- アライメント精度に悪影響

## 改善オプション

### オプション1: 固定範囲正規化（推奨）

位相画像の場合、範囲が既知（例: -1.1〜1.5 rad）なので固定範囲を使用：

```python
def to_uint8(img, vmin=-1.5, vmax=2.0):
    """固定範囲でuint8に変換"""
    clipped = np.clip(img, vmin, vmax)
    normalized = (clipped - vmin) / (vmax - vmin)
    return (normalized * 255).astype(np.uint8)
```

**メリット**:

- 全画像で一貫した変換
- 外れ値に強い
- シンプルで高速

### オプション2: パーセンタイルクリッピング

全画像の統計から範囲を決定：

```python
def to_uint8_percentile(img, pmin=1, pmax=99):
    """パーセンタイルでクリッピングしてuint8に変換"""
    vmin = np.percentile(img, pmin)
    vmax = np.percentile(img, pmax)
    clipped = np.clip(img, vmin, vmax)
    normalized = (clipped - vmin) / (vmax - vmin)
    return (normalized * 255).astype(np.uint8)
```

**メリット**:

- 外れ値を自動除外
- データに応じて適応

### オプション3: cv2.normalize使用

OpenCVの標準関数を使用：

```python
def to_uint8_cv2(img):
    """cv2.normalizeを使用"""
    normalized = cv2.normalize(img, None, 0, 255, cv2.NORM_MINMAX)
    return normalized.astype(np.uint8)
```

## 推奨実装

### ステップ1: 固定範囲版を実装

位相画像の場合、19_gaussian_backsub.pyで使用している範囲を参考：

- `hist_min = -1.1`
- `hist_max = 1.5`

安全マージンを含めて `-1.5` 〜 `2.0` を推奨

### ステップ2: パラメータ化

```python
def to_uint8(img, vmin=None, vmax=None, method='fixed'):
    """
    uint8に変換（複数の方法に対応）
    
    Parameters:
    -----------
    method : str
        'fixed' : 固定範囲 (vmin, vmax使用)
        'minmax' : 画像ごとのmin-max（現在の実装）
        'percentile' : パーセンタイルクリッピング
    """
    if method == 'fixed':
        if vmin is None or vmax is None:
            vmin, vmax = -1.5, 2.0  # デフォルト
        clipped = np.clip(img, vmin, vmax)
        normalized = (clipped - vmin) / (vmax - vmin)
    elif method == 'percentile':
        vmin = np.percentile(img, 1)
        vmax = np.percentile(img, 99)
        clipped = np.clip(img, vmin, vmax)
        normalized = (clipped - vmin) / (vmax - vmin)
    else:  # minmax (現在の実装)
        img_min = np.min(img)
        img_max = np.max(img)
        if img_max - img_min > 0:
            normalized = (img - img_min) / (img_max - img_min)
        else:
            normalized = np.zeros_like(img)
    
    return (normalized * 255).astype(np.uint8)
```
