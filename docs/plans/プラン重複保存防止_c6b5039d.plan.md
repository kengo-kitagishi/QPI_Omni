# プラン重複保存防止

## 問題
`notion_plan_save.py` は毎回無条件に新規ページを作成するため、同じプランを複数回実行すると重複ページが生成される。

## 修正内容

### [`scripts/notion_plan_save.py`](scripts/notion_plan_save.py)

1. `PATCH /databases/{DB_ID}/query` で `Name` が `[プラン] {title}` のページを検索
2. 見つかった場合 → スキップして既存ページの URL を表示して終了
3. 見つからない場合 → 通常通り新規作成
