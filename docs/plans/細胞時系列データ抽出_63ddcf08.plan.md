# 細胞時系列データ抽出スクリプト

## 📊 ユーザーの懸念への回答

Omniseggerのコードを調査した結果、以下が確認できました：

| 項目 | 使用されるもの | 詳細 |
|------|---------------|------|
| **内部判定の境界** | ✅ Omniposeのマスク | カスタムモデルのマスクがそのまま使用される |
| **体積計算** | ⚠️ Rod近似（代用） | 円柱+半球モデル: \( V = \pi R^2 (L - 2R) + \frac{4}{3}\pi R^3 \) |
| **蛍光平均計算** | ✅ マスク内の平均 | `総輝度 / マスク内ピクセル数` |

**結論**: 体積はRod近似（`Rrod`, `Lrod`）による代用的な推定値ですが、マスクは完全にOmniposeのものです。時系列変化の傾向を見るには十分です。

## 🎯 作成したスクリプト

### 1. [`extract_cell_timeseries.m`](omnisegger/extract_cell_timeseries.m)
**コア抽出関数**

- 細胞ファイル（`.mat`）から時系列データを抽出
- 体積、面積、長軸、短軸、Rodパラメータ、蛍光データを取得
- μm単位とピクセル単位の両方を出力

**使い方**:
```matlab
CONST = loadConstants('60XEc', false);
[time_data, info] = extract_cell_timeseries('F:/data/xy1/cell/cell001.mat', CONST);
```

**出力データ**:
- `time_data.volume_um3`: 体積（μm³）の時系列
- `time_data.fluor1_mean`: 蛍光1平均の時系列
- `time_data.time_min`: 時間（分）
- その他: 面積、長軸、短軸、Rodパラメータなど

---

### 2. [`quick_extract_and_plot.m`](omnisegger/quick_extract_and_plot.m)
**簡単に使えるラッパー関数**

- ワンコマンドでデータ抽出+6つのグラフ自動生成
- 体積、面積、長軸・短軸、Rod、蛍光のプロットを一括作成
- CSV保存オプション付き

**最もシンプルな使い方**:
```matlab
quick_extract_and_plot('F:/data/xy1', 1);
```

これだけで全て完了：
- ✅ データ抽出
- ✅ 6つのグラフ自動生成
- ✅ ワークスペースに保存（`time_data`, `cell_info`）
- ✅ CSV保存オプション

## 📈 取得できるデータ

```matlab
time_data = 
         frame: [n×1 double]  % フレーム番号
      time_min: [n×1 double]  % 時間（分）
    volume_um3: [n×1 double]  % 体積（μm³）
      area_um2: [n×1 double]  % 面積（μm²）
  longaxis_um: [n×1 double]  % 長軸（μm）
 shortaxis_um: [n×1 double]  % 短軸（μm）
       Rrod_um: [n×1 double]  % Rod半径（μm）
       Lrod_um: [n×1 double]  % Rod長さ（μm）
  fluor1_mean: [n×1 double]  % 蛍光1平均
   fluor1_sum: [n×1 double]  % 蛍光1総輝度
```
